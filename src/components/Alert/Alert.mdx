# Alert

An inline banner used to surface contextual feedback without interrupting the user's flow. Features a **2 px left-border accent**, **6 semantic variants**, an optional title, and a dismissible close button.

## Dismissible

Add `dismissible` to render a close (×) button on the right. Pair with `onClose` to handle the dismiss action.

```tsx
<Alert
  variant="info"
  description="Your session expires in 5 minutes."
  dismissible
  onClose={() => setVisible(false)}
/>
```

## Anatomy

```
┌──┬────────────────────────────────────────┬───┐
│  │  [icon]  [Title (optional)]            │ × │
│  │          [Description]                 │   │
└──┴────────────────────────────────────────┴───┘
 ↑ 2px left border                          ↑ close button (dismissible only)
```

| Slot | Details |
|------|---------|
| `icon` | Defaults to an info-circle SVG. Override with any `ReactNode` via the `icon` prop. |
| `title` | Optional bold heading — `AlertTitle` sub-component, `font-semibold text-sm`. |
| `description` | Required message text — `AlertDescription` sub-component, `font-normal text-sm`. |
| Close button | Rendered when `dismissible={true}`. Calls `onClose` on click. |

---

## Accessibility

- Root element uses `role="alert"` so screen readers announce the content immediately on mount.
- The close button has `aria-label="Dismiss alert"` and is keyboard-focusable.
- The default icon SVG is `aria-hidden="true"` — it is decorative; meaning comes from the text.
- Color alone is never the sole indicator of variant — each variant also differs in border and background tone.
