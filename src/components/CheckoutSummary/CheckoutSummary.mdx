## SubscriberEntry type

| Field | Type | Description |
|-------|------|-------------|
| `name` | `string` | Member's full name |
| `role` | `string` | e.g. `"Primary Member"` \| `"Dependent 1"` |
| `planName` | `string` | Dental plan name e.g. `"Espire Adult"` |
| `annualCost` | `string` | Formatted annual cost e.g. `"$292.42"` |
| `planCredit` | `string` | Formatted credit (negative) e.g. `"-$72.42"` |
| `isPrimary` | `boolean` | Uses sky-50 background + sky-200 divider |

## Accessibility

- All icon SVGs are `aria-hidden="true"` — meaning is conveyed by adjacent text.
- The T&C checkbox is a native `<input type="checkbox">` wrapped in a `<label>` — fully keyboard accessible.
- The "Have a Promo Code?" trigger is a semantic `<button>` with type="button".
- "Start Subscription" and "Save Draft" use the `Button` atom, which has built-in focus-ring and disabled state support.
