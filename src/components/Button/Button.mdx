# Button

A versatile action trigger supporting **6 visual hierarchies**, **4 sizes**, destructive states, icon slots, and icon-only mode. Built with CVA + Radix Slot so it renders as any element via `asChild`.

## Accessibility

- Renders as a native `<button>` by default — keyboard accessible out of the box.
- Icon-only buttons **must** include `aria-label` since visible text is hidden.
- `leadIcon` and `trailIcon` wrappers are `aria-hidden="true"` so screen readers skip the SVG markup.
- `asChild` delegates the accessible role to the child element (e.g. `<a>` renders as a link).
- Focus ring uses `focus-visible:ring-2` so keyboard users always see a clear indicator.
